# Global Superstore Sales Dashboard (Excel Project)

## Project Overview
This project analyzes global superstore sales data using Microsoft Excel.  
The goal is to understand sales performance, profit trends, category performance, and regional sales distribution.

## Tools Used
- Microsoft Excel
- PivotTables
- PivotCharts
- Slicers
- Data Visualization

## Dataset
The dataset contains sales transactions including:
- Order ID
- Sales
- Profit
- Discount
- Product Category
- Region
- Order Date

The dataset includes **51,000+ rows of sales records**.

## Dashboard KPIs
The dashboard tracks key business metrics:

- Total Orders: 51.3K
- Total Profit: 1.5M
- Total Sales: 12.6M
- Average Discount: 14.29%
- Profit Margin: 12%

## Analysis Performed

### Sales Trend Analysis
A line chart was used to analyze how sales changed over time.

### Profit Trend Analysis
Monthly profit trends were analyzed to understand business growth.

### Category Performance
Sales and profit were compared across categories:
- Technology
- Furniture
- Office Supplies

### Top Products
Top selling products were identified using bar charts.

### Regional Sales Distribution
Regional sales performance was analyzed using map visualization.

## Dashboard Features
- Interactive filters using slicers
- Dynamic pivot charts
- KPI indicators
- Category comparison
- Trend analysis 

## Conclusion
This project demonstrates how Excel can be used to create interactive dashboards and perform business data analysis.
